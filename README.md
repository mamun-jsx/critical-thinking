## Blog 1: Differences Between Interfaces and Types in TypeScript

An interface is the structure of an object. It is commonly used for object models, classes.

for an example `interface User {name:string, age:number}`. 

A types means the data formation what is it. is it string, object, function array or unions. 

for example `type User = {
  name: string;
  age: number;
};
`
